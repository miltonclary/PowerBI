<h1>Power BI AventureWorks Dashboard</h1>



<h2>Description</h2>
This adventure works Project consists of importing data into power BI, Transforming data to filter only the necesssary data using power query, creating joins from fact tables to be able to pull pertinent data from multiple tables, creating measures to perform various different aggregations to display in our visuals, various charts, tables, and data visuals, interactive filtering and drill down to show specific product information, and modifying the look and feel of the interactive dashboard by modifying the formatting.
<br />


<h2>Languages and Utilities Used</h2>

- <b>DAX</b> 
- <b>M</b>

<h2>Environments Used </h2>

- <b>Power BI</b> (21H2)
